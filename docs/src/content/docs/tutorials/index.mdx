## Before You Start

Make sure you have:

1. **Installed sensipy** - See the [Installation Guide](/getting_started/installation)
2. **Downloaded IRF files** - See the [Setup Guide](/getting_started/setup)
3. **Basic Python knowledge** - Familiarity with Python, numpy, and astropy units

## Tutorial Structure

Each tutorial follows this pattern:

- **Objective**: What you'll learn
- **Prerequisites**: Required data and setup
- **Step-by-step code**: Complete, runnable examples
- **Expected output**: What you should see
- **Further reading**: Links to comprehensive guides in [Getting Started](/getting_started/overview) and API references

<Aside type="tip">
  Tutorials focus on practical examples. For comprehensive documentation including

</Aside>
